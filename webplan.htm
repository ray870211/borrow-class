專題簡介

在這個系統中可以讓使用者進行教室的借用與查詢，此程式會把每個使用者與使用者借用的資訊記錄起來，其他使用者也可以透過查詢來了解目前教室的使用狀況，如果是老師，也可以簡單的利用編輯功能來刪除學生的借用紀錄。時間如果過了今天，系統會自動刪除過期的資料。

主要功能

  1. 會員註冊與登入登出
  2. 場地借用
  3. 教室查詢
  4. 編輯使用者

會員註冊與登入登出

  1. 註冊

    2. form表單，填寫後寫入資料庫。

    >
  2. 登入

    3. 利用SESSION實現登入功能

  3. 登出

    4. 刪除SESSION

場地借用

  1. 選擇教室與日期

    2. 利用POST與GET傳遞參數

  2. 寫入資料庫

場地查詢

  1. 選擇教室與日期

    2. 利用POST與GET傳遞參數

  2. 查詢資料庫

    3. 要根據每一個時間與大樓分別查詢

編輯使用者

  1. 查詢使用者

  2. 使用者借用的教室

資料庫

  1. class
  2. users
  3. 定期刪除資料事件

    4. 利用事件每天去去執行DELETE from class where(date< NOW())指令
